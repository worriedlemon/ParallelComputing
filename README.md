# Лабораторные работы по Архитектуре параллельных вычислительных систем

В этой дисциплине рассматриваются применения распараллеливающих библиотек для увеличения быстродействия алгоритмов в системе с общей памятью.

В качестве библиотек используются:
+ Microsoft MPI
+ OpenMP
